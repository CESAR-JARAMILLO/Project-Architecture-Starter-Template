# Folder Structure

This document explains how the codebase is organized.

## Root Structure

src
 ├ app
 ├ components
 ├ features
 ├ hooks
 ├ services
 ├ utils
 ├ types
 └ lib

---

## Directory Explanation

components
Reusable UI components.

features
Feature-based modules containing business logic and UI.

hooks
Custom React hooks.

services
API interaction layer.

utils
Reusable helper functions.

types
Shared TypeScript types.

lib
Core application utilities such as database and authentication.
